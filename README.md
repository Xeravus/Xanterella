# Xanterella
Mein Nixos-System besteht aus mehreren Host, die bestimmte Profile importieren, in denen bestimmte Packages deklariert sind.

## Hosts
Diese Host sind zurzeit zur Verfügung und eine kleine Erklärung, für jeden einzelnen:
1. Xeravus
    - Main Maschine
    - Hyprland
    - Nvidia & Intel
    - Remote Builder für die restlichen Hosts
2. Xorus
    - Altes Chromebook
    - Hyprland/Gnome(coming soon)
    - AMD
3. Xorus2(Neuer Name coming soon)
    - Alter Medion Laptop
    - Gnome(coming soon)
    - Intel
4. Vicuna
    - Home-Server
    - Raspberry Pi
    - Kein EM & WM
    - ARM
    - Host verwaltung(coming soon)
5. Installer
    - Installer/Live USB Stick
6. Crylia
    - Tempräre Nixos Umgebung
    - Wird vom Installer installiert
    - Reiner Übergang
